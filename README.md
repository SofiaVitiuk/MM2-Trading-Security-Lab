# MM2 Trading Security Lab 🎮🔒

Educational project teaching advanced backend security through trading system vulnerabilities.

## 📚 What You'll Learn
- ❌ How client-side manipulation breaks trades
- ✅ How server-side validation prevents it
- 🧪 How to test for vulnerabilities
- 🛡️ Best practices for secure systems

## 📂 Folders
- `vulnerable-version/` - BROKEN code (for learning)
- `secure-version/` - FIXED code (production-ready)
- `exploits/` - Exploit examples & tests
- `docs/` - Security analysis

## 🚀 Quick Start
1. Read `vulnerable-version/README.md` first
2. Study the insecure code
3. Try the exploits in `exploits/`
4. Compare with `secure-version/`

⚠️ **For educational use only - never attack real systems!**
